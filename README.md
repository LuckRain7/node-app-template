# node-app-template
提供基于 Express、Koa、Egg 框架的模板，用户可以基于此快速构建应用

- Express ^4.16.1
- Koa ^2.11.0
- Egg ^2.15.1

## Install

- Express

```bash
git clone https://github.com/LuckRain7/MyWorkLoad.git
cd Express
npm install
npm run dev
open http://localhost:3000
```

- Koa 

```bash
git clone https://github.com/LuckRain7/MyWorkLoad.git
cd Koa 
npm install
npm run dev
open http://localhost:3000
```
- Egg

```bash
git clone https://github.com/LuckRain7/MyWorkLoad.git
cd Egg
npm install
npm run start
open http://localhost:7001
```
